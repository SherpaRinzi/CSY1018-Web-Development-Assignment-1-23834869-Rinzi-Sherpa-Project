Namaste! My name is Rinzi Sherpa. I am the student of NAMI college(1st year). My course is BSc(Hons) Computing. This was our first assignment. This report is about our experience regarding the basic learning of web development. We were given to make a website and we had to create an a report on it.

Throughout this module I've faced a lot of ups and downs. So far we have learned to code using html and css.

Design development

In this module, I've created a simple website using html and css. The website focuses on showcasing the skills and experience, with a clean design that is easy to navigate and understand. we have used a simple fonts and a color combination that has a perfect match and easily readable. In the early stages of development, It was a challenge to understand the terms of web development, but I gradually became more comfortable with coding and debugging. Through online tutorials and classes, we learned how to structure and style web pages, as well as how to make responsive design.

My Experience

Overall, I found the module to be challenging, but also extremely rewarding. I gained a better understanding of web development and the process of coding, designing, and debugging. I learned how to structure web pages using HTML and style them using CSS. I also gained an understanding of the importance of designing for a positive user experience.
